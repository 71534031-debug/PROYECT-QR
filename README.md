# Sistema de Generación y Validación de Certificados con Código QR

## Descripción

Este proyecto es una aplicación web desarrollada para la generación, gestión y validación de certificados digitales mediante códigos QR.

El sistema permite administrar participantes, actividades, plantillas de certificados y verificar la autenticidad de los documentos emitidos mediante el escaneo de un código QR.

## Características principales

* Generación automática de certificados.
* Validación de certificados mediante código QR.
* Gestión de participantes.
* Gestión de actividades.
* Administración de plantillas de certificados.
* Panel de administración.
* API REST para la gestión de datos.
* Base de datos relacional.
* Pruebas unitarias e integración.


```bash

```

### Frontend

```bash
cd frontend
npm install

```
