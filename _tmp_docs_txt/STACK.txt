 Comandos Base
 Backend
# instalar dependencias
npm install

# modo desarrollo
npm run dev

# modo producción
npm start
Script sugerido en package.json:
"scripts": {
  "dev": "nodemon src/app.js",
  "start": "node src/app.js"
}
 Frontend
# instalar dependencias
npm install

# iniciar entorno desarrollo
npm run dev

# build producción
npm run build

# preview producción
npm run preview

Consideraciones Técnicas
Uso obligatorio de HTTPS en producción 
Validaciones críticas en backend 
Manejo de errores con códigos HTTP estándar 
Generación de PDF y QR debe ser asincrónica 
Soporte para carga masiva (CSV/Excel) 
Logs básicos para trazabilidad
